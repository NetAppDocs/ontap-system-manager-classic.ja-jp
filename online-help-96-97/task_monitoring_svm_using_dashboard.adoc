= System Manager - ONTAP 9.7 以前を使用して SVM を監視します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP System Manager クラシック（ ONTAP 9.7 以前で使用可能）のダッシュボードを使用すると、 Storage Virtual Machine （ SVM ）の健常性とパフォーマンスを監視できます。

.手順
. [ * ストレージ * > * SVMs * ] をクリックします。
. 監視する SVM を選択します。
. ダッシュボードパネル内の詳細情報を確認します。
