= System Manager classic - ONTAP 9.7 以前を使用して論理ストレージをセットアップします
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
論理ストレージは、 ONTAP System Manager クラシック（ ONTAP 9.7 以前で使用可能）を使用してセットアップできます。論理ストレージのセットアップでは、 Storage Virtual Machine （ SVM ）とボリュームを作成します。
