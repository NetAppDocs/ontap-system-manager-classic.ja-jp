= ソースボリュームを再アクティブ化
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ソースボリュームが使用できる状態になったら、デスティネーションボリュームからソースボリュームにデータを再同期し、再同期後の変更を反映して、ソースボリュームをアクティブ化する必要があります。
